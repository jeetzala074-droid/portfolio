4. Interdisciplinary Projects and Research

Project 1: Real-Time Logistics Optimization System

Disciplines: Computer Science (Graph Algorithms), Operations Research, Business Logistics

Objectives & Methodology:

To minimize last-mile delivery time and fuel consumption for a small local fleet. Implemented Dijkstra's and A* search algorithms, optimized for dynamic variables like traffic and weather, using Python and integrated with a simple web dashboard (React) for real-time visualization.

Outcomes & Critical Analysis:

The system demonstrated a **12% average reduction in route distance** during simulated peak hours. The main challenge was integrating live, noisy traffic data; this was solved by applying a Kalman filter to smooth the input data stream, enhancing the reliability of the predictive routing algorithm. This project showcased knowledge synthesis in discrete mathematics, systems integration, and economic efficiency.

Project 2: Housing Price Forecasting Model

Disciplines: Data Science, Econometrics, Machine Learning

Objectives & Methodology:

To predict property valuation with high accuracy using a diverse set of input features. Employed a **Gradient Boosting Regressor (XGBoost)** in Python, leveraging socio-economic indicators, geographical coordinates, and historical transaction data sourced from public APIs. Methodology included rigorous feature engineering and cross-validation.

Outcomes & Critical Analysis:

Achieved a **Root Mean Square Error (RMSE) of $15,000** on unseen test data, indicating strong predictive power. The innovation was the use of **Natural Language Processing (NLP)** to analyze local news sentiment (a soft factor) and incorporate it as a predictive feature, demonstrating the integration of soft and hard data types for complex problem-solving.

7. Global Awareness and Ethical Considerations

Case Study: Bias in Logistics Routing

**Scenario & Analysis:** In the Logistics Optimization Project (Section 4), the initial model showed a subtle bias, consistently favoring routes through low-income neighborhoods to achieve perceived speed gains, which would increase traffic and pollution in those areas. This presented a clear **ethical dilemma** concerning **algorithmic fairness** and environmental justice.

**Resolution & Reflection:** We modified the cost function to include a **"social impact penalty"** that dynamically factored in localized population density and air quality metrics alongside time and distance. This slowed the "optimal" route slightly but ensured equitable distribution of logistical burden. This experience reinforced my personal reflection on the responsibility of engineers to actively audit and mitigate the societal side-effects of their solutions.
